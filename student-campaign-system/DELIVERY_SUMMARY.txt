================================================================================
SCORING SYSTEM
================================================================================

Total Points: 100

1. Degree Relevance (0-30)
   - STEM: 30, Business: 25, Humanities: 20, Arts: 15, Other: 10

2. Objective Quality (0-25)
   - Specificity (10) + Feasibility (8) + Relevance (7)

3. Academic Standing (0-20)
   - Graduate: 20, Senior: 15, Junior: 12, Sophomore: 8, Freshman: 5

4. Demonstrated Need (0-15)
   - Financial Aid: 8, First Gen: 4, Underrepresented: 3

5. Claude Familiarity (0-10)
   - Experienced: 10, Moderate: 7, Beginner: 4, None: 0

Allocation Logic:
  Score ≥ Cutoff + Licenses Available → APPROVED
  Score ≥ Cutoff + No Licenses → WAITLISTED
  Score < Cutoff → REJECTED

Default Cutoff Scores:
  STEM: 60, Business: 55, Humanities: 50, Arts: 45, Other: 50

================================================================================
LICENSE MANAGEMENT
================================================================================

Default Period: 90 days
Max Extensions: 3 per license
Max Total Extension Days: 90 days
Extension Days per Report: 30 days
Progress Quality Threshold: 70/100

Progress Report Requirements:
  - Minimum 100 characters
  - Must include Claude usage examples
  - Must describe outcomes/deliverables
  - Can only be submitted within 30 days of expiration
